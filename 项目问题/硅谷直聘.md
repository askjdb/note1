# 硅谷直聘项目

## **1.1.** **项目描述**

+  此项目为一个前后台分离的招聘的 SPA, 包括前端应用和后端应用

+ 包括用户注册/登陆, 大神/老板列表, 实时聊天等模块

+  前端: 使用 React 全家桶+ES6+Webpack 等技术

+  后端: 使用 Node + express + mongodb + socketIO 等技术

+ 采用模块化、组件化、工程化的模式开发
